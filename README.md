# Buenas Viagens!

Landing page de uma agência de turismo, desenvolvida como trabalho da disciplina no Centro Universitário Farias Brito. O objetivo era reproduzir um design feito em Figma usando HTML, CSS e JavaScript puro.

Demo: https://mauriciodequeiroz.github.io/buenas-viagens/

Figma: https://www.figma.com/proto/K19kr99sFhEUqCg6UNdQyl/AvaliacaoBgTech?node-id=0-1&t=j9HY3VpP7sVJXzqL-1

Enunciado da avaliação: https://drive.google.com/file/d/1o8dK_DfldFV4X4FofpoPmxFipIpWBr2E/view?usp=drive_link

## Sobre o projeto

O site reproduz a landing page de uma agência de viagens fictícia a partir do design do Figma fornecido na avaliação. Foi construído sem frameworks, usando HTML semântico, CSS (Flexbox e Grid) e JavaScript puro para as partes interativas.

Funcionalidades implementadas:

- Navbar fixa com menu hambúrguer no mobile
- Hero com barra de busca de destino e datas
- Seção de próximos passeios com grade de fotos
- Cards de destinos com preço promocional e preço riscado
- Slider de avaliações de clientes (setas e também navegação pelo teclado)
- Formulário de newsletter com validação de e-mail
- Rodapé com links de parceiros e ícones de redes sociais
- Hover em links, botões e ícones
- Layout responsivo para mobile, tablet e desktop

Tecnologias: HTML5, CSS3, JavaScript, Google Fonts (Pacifico e Poppins), Font Awesome, Figma, Git/GitHub e GitHub Pages para o deploy.

## Como rodar localmente

```bash
git clone https://github.com/MauricioDeQueiroz/buenas-viagens.git
cd buenas-viagens
```

Depois é só abrir o `index.html` com a extensão Live Server no VS Code.

## Buenas Viagens! (English)

Tourism agency landing page, built as a university assignment for Centro Universitário Farias Brito. The goal was to reproduce a Figma design using plain HTML, CSS and JavaScript.

Demo: https://mauriciodequeiroz.github.io/buenas-viagens/

Figma: https://www.figma.com/proto/K19kr99sFhEUqCg6UNdQyl/AvaliacaoBgTech?node-id=0-1&t=j9HY3VpP7sVJXzqL-1

Assignment brief: https://drive.google.com/file/d/1o8dK_DfldFV4X4FofpoPmxFipIpWBr2E/view?usp=drive_link

### About

The site reproduces a fictional travel agency landing page from the Figma design provided in the assignment. Built without frameworks, using semantic HTML, CSS (Flexbox and Grid), and plain JavaScript for the interactive parts.

Implemented features:

- Fixed navbar with hamburger menu on mobile
- Hero section with a destination and date search bar
- Upcoming trips section with a photo grid
- Destination cards with promotional and crossed-out pricing
- Customer reviews slider (arrows and keyboard navigation)
- Newsletter form with email validation
- Footer with partner links and social media icons
- Hover states on links, buttons and icons
- Responsive layout for mobile, tablet and desktop

Tech: HTML5, CSS3, JavaScript, Google Fonts (Pacifico and Poppins), Font Awesome, Figma, Git/GitHub, and GitHub Pages for deployment.

### Running locally
